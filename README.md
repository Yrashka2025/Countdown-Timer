##  Installation

### Prerequisites
- Python 3.6 or higher

### Steps
1. Clone or download the repository
2. Navigate to the project directory
3. Run the script:
   ```bash
   python timer.py
   ```

##  How to Use

1. Start the timer by running `python timer.py`
2. Watch the countdown in real-time
3. Every 10 seconds, you'll be offered to play a mini-game
4. Type 'y' to play or 'n' to continue the countdown
5. Press Ctrl+C at any time to exit

##  Mini-Games

### Number Guessing Game
- Guess a number between 1 and 10
- You have 3 attempts
- Get hints: "too low" or "too high"

### Reaction Time Test
- Press ENTER when you see "GO!"
- Measures your reaction time in milliseconds

##  Dependencies

The script uses only Python standard library modules:
- `time`
- `datetime`
- `random`
- `os`

No external dependencies required!
